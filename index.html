Un Hosting es una computadora física que funciona sin interrupciones para que su sitio web esté disponible todo el tiempo para cualquier persona que quiera verlo. Su proveedor de alojamiento web es responsable de mantener ese servidor en funcionamiento, protegerlo de ataques maliciosos y transferir su contenido, como texto, imágenes, archivos, etc., desde el servidor a los navegadores de sus visitantes.

Cuando decide iniciar un nuevo sitio web, necesita encontrar un proveedor de alojamiento web que le proporcione ese espacio de servidor. Su servidor web almacena todos sus archivos, activos y bases de datos en el servidor. Cada vez que alguien escribe su nombre de dominio en la barra de direcciones de su navegador, su anfitrión transfiere todos los archivos necesarios para atender esa solicitud.

Por lo tanto, debe elegir el plan de alojamiento que mejor se adapte a sus necesidades. De hecho, funciona de manera similar a los alquileres de viviendas: debe pagar el alquiler con regularidad para mantener el servidor en funcionamiento continuamente.



Un Dominio es como un alias que permitirá a cualquier usuario de la Web localizar nuestra pagina o Sitio Web. Cambien se utiliza con correo electrónico. Internamente los servidores web utilizan una serie de cuatro números separados por puntos (IP) para comunicarse entre si. El nombre de dominio es el “alias” que sustituye a esos números porque siempre es mas fácil recordar un nombre que una serie de números.

El aspecto mas importante de un dominio es su inocuidad; es decir, es completamente imposible que dos personas, empresas u organizaciones tengan el mismo nombre de dominio simultáneamente. De este modo, el dominio se convierte en una autentica señal de identidad en la Web. Es una parte imprescindible de la identidad corporativa de muchas empresas e instituciones.


Dominios genéricos

.ar para servicios de Argentina.

.es para servicios de España.

.biz previsto para ser usado en negocios.

.gov y .gob para gobierno y entidades públicas.

.info para información.

.org para organizaciones.


Dirección IP (Dirección de protocolo de Internet). La forma estándar de identificar un equipo que está conectado a Internet, de forma similar a como un número de teléfono identifica un número de teléfono en una red telefónica. La dirección IP consta de cuatro números separados por puntos y cada numero es menor de 256; por ejemplo 192.200.44.69. El administrador del servidor Web o su proveedor de servicios de Internet asignará una dirección IP a su equipo.


FTP (Protocolo de transferencia de archivos). Es uno de los protocolos más antiguos en uso hoy en día, y es una forma conveniente de mover archivos. Un servidor FTP ofrece acceso a un directorio con subdirectorios. Los usuarios se conectan a estos servidores con un cliente FTP, una pieza de software que les permite descargar archivos del servidor, así como cargar archivos en él.


WordPress Se trata de una aplicación de software para crear y gestionar sitios web (crear sus contenidos, etc.), tiene la peculiaridad de ser un proyecto de software libre y de fuentes abiertas que depende de la Fundación WordPress. Esto quiere decir que su código es creado y mantenido por una comunidad de miles de desarrolladores voluntarios (lo que garantiza su continuidad) y que puedes disponer de la aplicación completamente gratis.


LINK/URL (Localizador uniforme de recursos) es la dirección de internet que emplea para acceder a una pagina web, un servidor web o cualquier otro dispositivo existente en la red. En teoría cada URL valida apunta a un único recurso. Dichos recursos pueden ser páginas HTML, documentos CSS, imágenes etc.
